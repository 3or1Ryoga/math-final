🪙 開始前にリロードする「型」

  • 特異行列の写像：$A\boldsymbol{x}=(\,\text{スカラー}\,)\times(\text{列ベクトル})$ に分解。$\mathrm{Im}$＝列の直線、$\mathrm{Ker}$＝$A\boldsymbol{x}=\boldsymbol0$ の直線。
  • 一次結合の罠：基底が一次従属（$\det=0$）だと、$\boldsymbol a$ は表せるが $\boldsymbol b$ は矛盾＝表せない。3年連続。
  • Im/Ker：掃き出し→rank $r$。$\dim\mathrm{Im}=r$（基底は元の列）、$\dim\mathrm{Ker}=n-r$。次元定理 $\dim\mathrm{Im}+\dim\mathrm{Ker}=n$。
  • グラム・シュミット：既に作った $\boldsymbol e_i$ 方向の射影を引いて正規化、を繰り返す。$\boldsymbol u_k=\boldsymbol0$ なら一次従属。
  • 対角化：$\det(A-\lambda I)=0$→固有ベクトルを列に並べ $P$、$P^{-1}AP=\mathrm{diag}(\lambda_i)$。検算は $\sum\lambda=\mathrm{tr}A$, $\prod\lambda=\det A$。
  • 直交補空間：$V$ が「$A\boldsymbol x=\boldsymbol0$ の解空間」なら $V^\perp$＝係数（行）ベクトルが張る空間。$\dim V+\dim V^\perp=n$。

① 写像プロット + Im/Ker P1・毎年出る

表現行列 $A$ で図の点がどこへ写るかプロットし、$\mathrm{Im}f$・$\mathrm{Ker}f$ を図示。過去 $A$ は3年とも $\det A=0$ の特異行列。

解き方テンプレ

  1. 各点 $(x,y)$ に $A$ を掛けて写像先を計算し、矢印つきでプロット（写像前後の両方を描く）。
  2. $\det A=0$ なら平面全体が1本の直線につぶれる。$\mathrm{Im}f$＝列ベクトルの張る直線。
  3. $\mathrm{Ker}f$＝$A\boldsymbol x=\boldsymbol0$ の解＝1行目 $ax+by=0$ の直線。
  4. 25年以降は「次元と一組の基底」も明記要求 → $\dim\mathrm{Im}=1$, $\dim\mathrm{Ker}=1$ と基底ベクトルを書く。

② 連立1次方程式の一般解 P1・隔年で出る

一般解を「特殊解 ＋ 同伴な同次方程式の基本解の1次結合」の形で書く。形式指定を守るのが得点のカギ。

解き方テンプレ

  1. 拡大係数行列を掃き出しで階段行列に。
  2. 先頭1が立たない列の変数（自由変数）に $s,t,\dots$ を置く。
  3. $\boldsymbol x=\boldsymbol x_0+s\boldsymbol u_1+t\boldsymbol u_2$ に整理。$\boldsymbol x_0$＝特殊解、$\boldsymbol u_i$＝基本解（本数＝自由変数の数＝$n-\mathrm{rank}$）。
  4. 検算：$\boldsymbol x_0$ を代入して右辺一致、$\boldsymbol u_i$ を代入して $0$。

③ 一次結合で表す P1・毎年＋罠

$\boldsymbol a,\boldsymbol b$ を $\boldsymbol a_1,\boldsymbol a_2,\boldsymbol a_3$ の一次結合で表す。基底はわざと一次従属にしてあり、片方は「表せない」が正解。

観察が最速

$\boldsymbol a_1,\boldsymbol a_2,\boldsymbol a_3$ はどれも第1・2成分が $(1,-1)$。よってどんな結合も「第2成分 $=$ −第1成分」になる。

⑥ 直交補空間 $V^\perp$ P1・24-25連続

条件式で定義された部分空間 $V$ の直交補空間を求める。$V$ が「$A\boldsymbol x=\boldsymbol0$」型なら係数ベクトルが張る空間がそのまま $V^\perp$。

解き方テンプレ

  1. $V$ をパラメータ表示して基底を出す。
  2. $V^\perp=\{\boldsymbol y:\boldsymbol y\cdot\boldsymbol v=0\ (\forall \boldsymbol v\in V\text{の基底})\}$ を連立で解く。
  3. ショートカット：$V=\{A\boldsymbol x=\boldsymbol0\}$ なら $V^\perp=$ 行ベクトルが張る空間。
  4. 次元チェック $\dim V+\dim V^\perp=n$。

✍️ 係数ベクトル2本がそのまま $V^\perp$ の基底

$V$ を求める

$V=\langle(2,3,2)\rangle,\quad \dim V=1$

$V^\perp$（ショートカット）

$V^\perp=\langle(2,-2,1),\ (1,-2,2)\rangle,\quad \dim V^\perp=2$
検算：$(2,-2,1)\cdot(2,3,2)=4-6+2=0$ ✓、$(1,-2,2)\cdot(2,3,2)=2-6+4=0$ ✓。$1+2=3$ ✓。

⑦ 行列の対角化 P1・2025新出

固有値・固有ベクトルを求め、変換行列 $P$ で対角化する。2025年に新登場したので2026も要警戒。

解き方テンプレ

  1. 固有方程式 $\det(A-\lambda I)=0$ を解く（3次は整数解を代入で探す）。
  2. 各 $\lambda_i$ で $(A-\lambda_i I)\boldsymbol x=\boldsymbol0$ を解いて固有ベクトル $\boldsymbol p_i$。
  3. 固有値が全て異なれば対角化可能（重解は固有ベクトルが次元分とれるか確認）。
  4. $P=[\boldsymbol p_1\,\boldsymbol p_2\,\boldsymbol p_3]$ とすると $P^{-1}AP=\mathrm{diag}(\lambda_1,\lambda_2,\lambda_3)$。
  5. 検算は $A\boldsymbol p_i=\lambda_i\boldsymbol p_i$（掛けるだけ）＋ $\sum\lambda=\mathrm{tr}A$, $\prod\lambda=\det A$。
2025年度 問5過去問（実物）新出
$A=\begin{pmatrix}6&4&-4\\-3&-2&3\\5&4&-3\end{pmatrix}$ が対角化可能ならば変換の行列を求めて対角化せよ。
✍️ 固有値 → 各固有ベクトル → $P$ を組む → 検算

固有値

$\det(A-\lambda I)=-(\lambda-1)(\lambda-2)(\lambda+2)=0\ \Rightarrow\ \lambda=1,2,-2$
3つとも相異なるので対角化可能。検算：$1+2-2=1=\mathrm{tr}A\,(6-2-3)$ ✓。

固有ベクトル

$\lambda=1$: $(A-I)\boldsymbol x=0$ → $x_1=0,\ x_2=x_3$ → $\boldsymbol p_1=(0,1,1)$
$\lambda=2$: $(A-2I)\boldsymbol x=0$ → $x_2=0,\ x_1=x_3$ → $\boldsymbol p_2=(1,0,1)$
$\lambda=-2$: $(A+2I)\boldsymbol x=0$ → $x_3=x_1,\ x_2=-x_1$ → $\boldsymbol p_3=(1,-1,1)$

結論

$P=\begin{pmatrix}0&1&1\\1&0&-1\\1&1&1\end{pmatrix}\ \Rightarrow\ P^{-1}AP=\begin{pmatrix}1&0&0\\0&2&0\\0&0&-2\end{pmatrix}$
検算（掛け算のみ）：$A\boldsymbol p_1=(0,1,1)=1\boldsymbol p_1$ ✓、$A\boldsymbol p_2=(2,0,2)=2\boldsymbol p_2$ ✓、$A\boldsymbol p_3=(-2,2,-2)=-2\boldsymbol p_3$ ✓。
対角成分の並びは $P$ の列の順番どおり。ズレると減点。
理解度:

🪙 ALA 公式チートシート

  • ノルム/距離/角度：$\|\boldsymbol x\|=\sqrt{\boldsymbol x^T\boldsymbol x}$、距離 $=\|\boldsymbol a-\boldsymbol b\|$、$\theta=\arccos\frac{\boldsymbol a^T\boldsymbol b}{\|\boldsymbol a\|\|\boldsymbol b\|}$、$\|\boldsymbol a\pm\boldsymbol b\|^2=\|\boldsymbol a\|^2\pm2\boldsymbol a^T\boldsymbol b+\|\boldsymbol b\|^2$。
  • QR分解：$A=QR$、$Q^TQ=I$（列が正規直交＝GSの結果）、$R$ は上三角。検算問題は $QR=A$ と $Q^TQ=I$ を掛けて確認。
  • 擬似逆行列（縦長・列独立）：$A^\dagger=(A^TA)^{-1}A^T=R^{-1}Q^T$（左逆）。横長は $A^\dagger=A^T(AA^T)^{-1}$（右逆）。
  • 最小二乗：$\min\|A\boldsymbol x-\boldsymbol b\|^2$ の解は $\hat{\boldsymbol x}=(A^TA)^{-1}A^T\boldsymbol b=A^\dagger\boldsymbol b$（正規方程式 $A^TA\hat{\boldsymbol x}=A^T\boldsymbol b$）。
  • 状態遷移：$\boldsymbol x_{t+1}=A\boldsymbol x_t$ → $\boldsymbol x_3=A^2\boldsymbol x_1$（感染症 fig9.5）。
  • ムーアの法則（ex13.3）：$\log_{10}y\approx\theta_1+\theta_2(t-1970)$ を最小二乗で当てはめ。$\theta_2\approx0.15$ → 約2年で2倍。
